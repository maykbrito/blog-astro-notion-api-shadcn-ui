Eu precisei de algo parecido em um projeto usando Astro…
Supondo que o componente receba 100 `posts`.


`index.astro`


```javascript
---
import Posts from '@/components/Posts.svelte'
const posts = [] // 100 posts 
---

<Posts posts={posts} />
```


`Posts.svelte` 


```javascript
<script>
  export let posts;
  let currentItems = 2;
</script>

{#each posts.slice(0, currentItems) as post}
	<h2>{post.title}</h2>
{/each}

{#if currentItems < posts.length}
  <button
    on:click={() => currentItems = currentItems + 2}
    type="button">
    Carregar Mais
  </button>
{/if}
```
